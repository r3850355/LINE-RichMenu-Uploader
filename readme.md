為了快速上傳RichMenu而誕生的工具

只要填好資料然後執行

就這麼簡單

## 安裝
```
npm install
```

## 設定
```
// index.js
{
  accessToken: '', // Channel AccessToken
  richMenuData: {}, // rich menu object
  imagePath: '' // 圖片路徑
}
```

## 執行
```
npm start
```
